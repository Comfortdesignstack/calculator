# Calculator

A responsive calculator application built with **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean and user-friendly interface.

## 🚀 Features

* Addition, subtraction, multiplication, and division
* Clear and delete functionality
* Decimal number support
* Responsive design for mobile and desktop screens
* Interactive buttons with JavaScript functionality

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
